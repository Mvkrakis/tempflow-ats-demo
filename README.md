# TempFlow ATS — Public Portfolio Demo

A static, privacy-safe portfolio demo of an automated ATS for a Junior IT Technician Associate role.

## Safe by design

- All applicant names, email addresses, CV names, scores, and analytics are fictional.
- No CV upload, database, email, API key, password, or personal data exists in this repository.
- The dashboard is an interactive front-end demo only.

## Run locally

Open `index.html` in a browser, or use the VS Code Live Server extension.

## Publish on GitHub Pages

1. Create a new **public** GitHub repository.
2. Upload these files to its root.
3. In GitHub, go to **Settings → Pages**.
4. Select **GitHub Actions** as the source.
5. Push to `main`; the included workflow publishes the demo.

The private/full implementation belongs in a separate repository or deployment with environment secrets, authenticated access, and encrypted CV handling.
